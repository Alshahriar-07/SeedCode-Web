Getting Started

Install Seed Code with pip or uv, then launch it from any terminal. The first run walks you through connecting a provider.

$ pip install seedcode-cli && seedcode

Prefer an isolated install or a Windows setup wizard? See the download page for every method.

Commands

Six commands cover the entire workflow. Each one is fast, predictable, and beautifully rendered.

seedcode

Launch the interactive assistant in your current directory.

$ seedcode
Welcome back. Connected to OpenRouter · GLM 5.2
seedcode chat

Start a focused chat session with your configured model.

$ seedcode chat
Chat session started. Type your prompt to begin.
seedcode config

Open the configuration wizard for providers, keys, and themes.

$ seedcode config
Config loaded from ~/.seedcode/config.toml
seedcode doctor

Diagnose your setup — keys, connectivity, and environment.

$ seedcode doctor
✓ 6 checks passed · environment healthy
seedcode models

Browse and search every model available on your provider.

$ seedcode models
312 models available · 47 free
seedcode update

Update Seed Code CLI to the latest release in place.

$ seedcode update
Already on the latest version (v1.0.0)

Configuration

Seed Code stores its settings in a single TOML file at ~/.seedcode/config.toml. Run seedcode config for the guided wizard, or edit the file directly.

config.toml
[provider]
name = "openrouter"
model = "glm-5.2"
stream = true
 
[appearance]
theme = "midnight"
markdown = true

API keys

Keys are stored securely by the config wizard. For CI and containers, the SEEDCODE_API_KEY environment variable takes precedence over the config file.

Troubleshooting

Start with the built-in diagnostics — it checks your Python version, API key, network, and terminal capabilities in one pass.

$ seedcode doctor

Still stuck? Check the FAQ or open an issue on GitHub — the maintainers respond quickly.
