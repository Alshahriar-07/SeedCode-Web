Install methods

Three ways in. All of them fast.

pip

Method 1

The classic route. Requires Python 3.9+ and installs straight from PyPI.

$ pip install seedcode-cli

uv

Method 2

Installed as an isolated tool — instant updates and no environment pollution.

$ uv tool install seedcode-cli

Windows Installer

Method 3

A standalone setup wizard for Windows. No Python installation required.

SeedCodeSetup.exe

Verify your installation

One command confirms everything is in place.

$ seedcode --version

Expected output: seedcode v1.0.0. Something off? Run seedcode doctor or see the FAQ.
