Installation

Up and running in seconds

Three ways to install. One command to verify. Zero configuration required to start.

pip

Method 1

Install from PyPI with Python 3.9 or newer.

$ pip install seedcode-cli

uv

Method 2

Install as an isolated tool with uv ‚ÄĒ fast and clean.

$ uv tool install seedcode-cli

Windows Installer

Method 3

A standalone installer ‚ÄĒ no Python required.

SeedCodeSetup.exe
Verify your install with seedcode --version ‚ÄĒ you should see v1.0.0.
